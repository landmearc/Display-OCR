# Display-OCR
v3.1.15
画面の中を何でもOCRして任意のアプリへ文字列入力するアプリ
画面スキャンで表示されているPDFやアプリに表示されている文字列を矩形選択すると、OCR機能で文字列に変換します
変換が誤っている場合は修正して保存することで学習します
文字列を張り付けたいアプリの入力欄へフォーカスした後、「この文字列をペースト」「選択した履歴をペースト」「選択した提携文字列をペースト」で任意のアプリへ文字列を入力することが出来ます。
不動産登記規則第93条ただし書き調査報告書へ登記事項証明書から転記するのに役立ちます。
